Base Endpoint
POSThttps://koda-da05205d.base44.app/functions/sendEmail
Send Email
Send transactional emails via Resend API. Powered by Clincoo backend.
Request Body
{ "to": "recipient@example.com", "subject": "Hello from Clincoo", "html": "<p>Your email content</p>", "from": "Name <noreply@resend.dev>" // optional }
Example — JavaScript
const res = await fetch('https://koda-da05205d.base44.app/functions/sendEmail', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ to: 'user@example.com', subject: 'Order Confirmed!', html: '<h1>Thanks for your order!</h1>' }) }); const data = await res.json(); console.log(data); // { success: true, id: "..." }
Example — cURL
curl -X POST https://koda-da05205d.base44.app/functions/sendEmail -H "Content-Type: application/json" -d '{"to":"user@example.com","subject":"Hello","html":"<p>Hi!</p>"}'
